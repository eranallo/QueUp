# ✨ QueUp

**Your ultimate trip planner for Walt Disney World & Universal Orlando Resort.**

Built with React + Vite. All data is stored locally in `localStorage` — no account needed.

## Features

- 🎢 **Ride Encyclopedia** — Every attraction at all 7 parks with history, trivia, specs & thrill ratings
- 📋 **Day Planner** — Check off every ride across all parks with progress tracking
- 🐭 **Hidden Mickey Tracker** — 31 locations with difficulty ratings and descriptions
- 🍹 **Food & Drink Tracker** — EPCOT Drinking Around the World challenge + Disney & Universal must-eats
- 💾 **Auto-saves** — All your progress is stored in your browser automatically

## Parks Covered

**Walt Disney World:**
- 🏰 Magic Kingdom (27 attractions)
- 🌍 EPCOT (13 attractions)
- 🎬 Hollywood Studios (11 attractions)
- 🦁 Animal Kingdom (9 attractions)

**Universal Orlando:**
- 🎬 Universal Studios Florida (9 attractions)
- ⚓ Islands of Adventure (11 attractions)
- ✨ Epic Universe — Opened May 2025 (7 attractions)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy to Vercel

1. Push this repo to GitHub
2. Import into [Vercel](https://vercel.com)
3. Framework: **Vite** — Vercel auto-detects this
4. Deploy 🚀

No environment variables needed.

## Tech Stack

- React 18
- React Router v6
- Vite
- Pure CSS (no Tailwind)
- localStorage for state persistence
- Google Fonts: Cinzel + Cormorant Garamond + Nunito
